# 1차 스터디 산출물 코니(Connie) — 중간 보고서

**사내 지식 가이드 챗봇 | Company Helper & AI Virtual Interactive Service**

---

## 1. 프로젝트 개요

### 1.1 목적

- 신규 입사자의 반복적인 기초 정보(위치, 규정) 문의로 인한 업무 몰입 저해 해소
- 사내 가이드 문서 파편화로 인한 정보 검색 시간 단축
- 온보딩 프로세스 자동화 및 HR/총무팀 반복 문의 응대 시간 절감

### 1.2 핵심 가치

- **사내 기본 정보 전달** 
- **RAG 기반 지능형 응답**: ChromaDB 벡터 검색 + Ollama LLM 활용
- **실시간 지식 업데이트**: 어드민에서 즉시 지식 보강 가능

### 3.4 답변 파이프라인

```
[질문 입력]
    ↓
[1단계] Exact Match (키워드 포함 여부)
    → 매칭 시 → 즉시 답변 + refLink + 첨부파일
    ↓ (미매칭)
[2단계] ChromaDB 벡터 검색 (Top 5)
    → 결과 있음 → Ollama RAG 답변 생성
    → 결과 없음 → Ollama 일반 지식 답변 (disclaimer 표시)
```

## 6. 실행 방법

```bash
# 1. ChromaDB (Docker)
docker run -d -p 8000:8000 chromadb/chroma

# 2. Ollama
ollama serve &
ollama pull llama3

# 3. 서버
cd server && npm start

# 4. 클라이언트 (별도 터미널)
cd client && npm run dev
# → http://localhost:5173
```

**통합 실행** (루트에서):

```bash
npm run dev   # 서버(3001) + 클라이언트(5173) 동시 실행
```
